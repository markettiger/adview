# 광고 데이터 대시보드 (PC)

메타 애즈 / 애플 애즈 데이터를 한 화면에서 확인하는 정적 대시보드입니다.

## 요구사항 반영

1. **메타/애플 API 연동 지원**
   - `API Base URL` + 각 플랫폼 `광고 ID`를 입력하면 아래 엔드포인트로 요청합니다.
     - `GET /api/meta-ads?adId={META_AD_ID}`
     - `GET /api/apple-ads?adId={APPLE_AD_ID}`

2. **일/주/월 집계 표 + 그래프**
   - 기간 단위 토글로 일간/주간/월간 집계를 전환합니다.
   - 표에는 주요 지표(노출/클릭/전환/비용/CTR)가 표시됩니다.
   - 그래프는 집계 단위별 Impressions 추이를 보여줍니다.

3. **왼쪽 사이드바 필터**
   - 전체 / 메타 / 애플 데이터 필터를 제공합니다.

## API 응답 형식 예시

```json
{
  "data": [
    {
      "date": "2026-04-01",
      "impressions": 12000,
      "clicks": 580,
      "conversions": 33,
      "spend": 320000
    }
  ]
}
```

## 실행

브라우저에서 `index.html`을 열면 됩니다.

- 실제 API를 연결하지 않아도, `샘플 데이터 보기` 버튼으로 UI 검증 가능
- CORS 정책이 있는 경우 API 서버에서 허용 설정 필요
